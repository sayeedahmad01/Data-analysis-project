# Employee Salary Prediction using Linear and Polynomial Regression

This project compares Linear Regression and Polynomial Regression models on an employee salary dataset.

## Features

* Data Visualization
* Linear Regression
* Polynomial Regression
* Salary Prediction

## Libraries Used

* NumPy
* Pandas
* Matplotlib
* Scikit-learn

## How to Run

```bash
pip install -r requirements.txt
python data_analysis.py
```
