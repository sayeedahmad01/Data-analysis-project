# Countdown Timer with Desktop Notification

A Python GUI application built using Tkinter and Plyer.

## Features

* Set hours, minutes, and seconds
* Live countdown display
* Desktop notification when timer finishes
* Popup alert after completion
* Simple and user-friendly interface

## Technologies Used

* Python
* Tkinter
* Plyer

## Installation

```bash
pip install plyer
```

## Run

```bash
python timer.py
```
